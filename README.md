# 🤖 BOTLEAGUE

India's Ultimate Robotics Arena. A React web application to track tournament brackets, manage community registrations, and view pilot rankings.

## 🚀 Features

- **Live Brackets & Results**: Tournament brackets, next matches, and event archives.
- **Registration**: Setup workflows for Judges, Volunteers, and Community Pilots.
- **Leaderboards**: Stands showing ranks, wins, win rates, and league points.
- **Divisions & Roadmap**: Tournament visual pathways mapped by age groups.

## 🛠️ Tech Stack

- React 19, Vite, Tailwind CSS
- Redux Toolkit, TanStack React Query

## 💻 Quick Start

```bash
npm install   # Install dependencies
npm run dev   # Start development server
npm run build # Build for production
npm run lint  # Run ESLint
```
